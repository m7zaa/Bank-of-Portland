# _Bank Account_

#### _Bank account project, 2019_

#### By _**Neha & Micah**_

## Description

_In this project, a user can open an account then deposit and withdraw "money"._

## Setup/Installation Requirements

* _Open in url_



## Specs
| Behavior | Input | Output |
| ------------- |:-------------:| -----:|
| User can make account with name and initial deposit. | Micah, $400 | Prompt:"You have successfully opened an account" |
| User can view balance | 400 | 400 |
| User can make another deposit. Deposit amount will be added to balance. | +$200 | New balance: $600 |
| User can make a withdrawal. Withdrawal amount will be deducted from balance. | -$200 | New balance:$600 |
